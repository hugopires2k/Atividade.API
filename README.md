# 🚀 Projeto Rick and Morty: Consumo de API

Este projeto foi desenvolvido como um desafio prático para a disciplina de Engenharia de Software. A ideia principal foi aprender a ligar o "mundo real"
(dados de uma API) ao design visual que o utilizador final utiliza.

## 📝 Sobre o Projeto
Em vez de escrever manualmente os dados dos personagens, esta aplicação "conversa" com um servidor externo e monta a página automaticamente.
O objetivo foi criar uma experiência leve, simples e totalmente pensada para quem acede pelo telemóvel.

## 🛠️ O que foi utilizado
* **HTML & CSS:** Para criar uma interface limpa e responsiva.
* **JavaScript:** Para fazer a busca dos dados de forma assíncrona.
* **API do Rick and Morty:** A fonte de toda a informação dos personagens.

## 💡 O que este projeto representa
Mais do que apenas código, este trabalho reflete o meu aprendizado em como transformar dados técnicos numa interface amigável e funcional para qualquer pessoa.
